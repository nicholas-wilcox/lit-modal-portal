Class ConfirmModal

A stock custom element <confirm-modal> for a confirmation dialog modal. The content of the confirmation prompt is exposed via a <slot>, and the cancel/confirm buttons are displayed below. Support for a secondary, alternative action is included.

There's a reactive property that controls whether the modal closes upon clicking the confirmation button, which defaults to true.

This component exposes [[LitDialog.enableLightDismiss]] and makes it default to true.
