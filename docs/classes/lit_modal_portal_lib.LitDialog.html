Class LitDialog

A wrapper around the <dialog> element that hooks into the [[ModalPortal]] API. The dialog's showModal() function is called after this component is first rendered in the DOM. The "removeModal" event is triggered whenever the dialog closes.

This custom element resets some of the default styles of <dialog> and effectively makes its <dialog> a fixed flexbox container with all 0 insets. In other words, it acts as a container for whatever is the content of the modal. Whatever is put into this component's <slot> is responsible for having a border, a fully opaque background, etc.
